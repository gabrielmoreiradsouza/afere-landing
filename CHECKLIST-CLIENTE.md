# Checklist de coleta — Landing Page Jornada Ads PRO

Lista do que precisa estar em mãos antes de começar a customizar o template
para um novo cliente.

## Identidade visual

- [ ] Logo PNG com fundo transparente (alta resolução)
- [ ] Logo em variação alta + horizontal (preferida)
- [ ] Cor primária da marca (hex)
- [ ] Cor secundária / neutra (hex)
- [ ] Fontes (se já tiver guideline; senão usamos Inter + Space Grotesk)

## Conteúdo textual

- [ ] Nome comercial e razão social
- [ ] Tagline / slogan (se houver)
- [ ] **Headline principal** (uma frase, ≤ 60 caracteres)
- [ ] **Palavra-destaque** dentro da headline (vai em verde neon)
- [ ] **Subheadline** com promessa/diferencial (1-2 linhas)
  - [ ] Trecho a sublinhar (a "promessa")
- [ ] Promessa numérica chave (ex: "10 dias", "24h", "−5 dias vs mercado")

## Produtos / Serviços (6 blocos)

Para cada produto:
- [ ] Nome
- [ ] Kicker emocional (uma palavra: Resistência, Precisão, Qualidade…)
- [ ] Descrição 2-3 linhas
- [ ] Foto (se não tiver, banco)
- [ ] Mensagem específica de WhatsApp
  (ex: "Olá! Quero orçamento de chumbadores.")

## Diferenciais (4 blocos)

Para cada um: ícone + título curto + 1 frase explicativa.

## Processo / Como funciona

5 passos do contato ao entregue. Cada passo: nome curto + 1 frase.

## Prova social

- [ ] 1 foto destaque (obra/cliente) + texto explicativo
- [ ] 1 depoimento (frase + autor)
- [ ] Lista de setores/clientes atendidos

## Qualificação no formulário

- [ ] **Pergunta 1** (4 opções) — geralmente fase do projeto/obra
- [ ] **Pergunta 2** (4 opções) — geralmente faturamento ou volume
- [ ] Definir quais combinações **passam** vs **não passam** na triagem

## FAQ (5 perguntas)

Perguntas frequentes que costumam barrar a venda.

## Dados institucionais (footer)

- [ ] Razão social + CNPJ
- [ ] Endereço completo
- [ ] WhatsApp comercial (formato E.164: 5541999090589)
- [ ] WhatsApp display (formato BR: (41) 9 9909-0589)
- [ ] E-mail (se quiser exibir)
- [ ] Redes sociais (se tiver)

## Domínio e deploy

- [ ] Subdomínio definido (ex: `cliente.moreirads.cloud`)
- [ ] Repositório GitHub criado
- [ ] DNS apontado para Hostinger

## Tracking

- [ ] GTM ID
- [ ] Meta Pixel ID
- [ ] Conta de anúncios com agência adicionada
